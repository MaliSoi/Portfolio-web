Swap Now | App

Brief

"Swap Now" is an e-store concept for swapping, selling, and buying second-hand clothing within trusted social groups (friends, family) and a close radius. A key future feature is to facilitate community events and pop-up fairs, further fostering a stronger sustainable fashion community.

Timeline: 2 months
Service: UX/UI designer
Role: Principal Designer
Tools: Figma, Photoshop, Illustrator.

Swap Now app Mockup

The Challenge

The high cost of clothing in Argentina created a need for accessible second-hand options. The challenge was to create a secure, direct, and community-driven online channel for trusted groups to exchange items, bridging the gap between informal chats and secure platforms.

The Process

  • Research: Identified user needs for secure group-based exchanges and circular fashion interest.
  • Ideation: Developed initial Sketches and defined a clear User Flow.
  • Prototyping: Created Mid-Fidelity Wireframes, iteratively refining designs based on usability considerations (e.g., improved transaction steps, simplified navigation icons).

Galería

User Flow: Detailing the intuitive path users take to browse, connect, and exchange items on Swap Now.

Sketches: The foundational stage of design, outlining initial screen ideas and user interactions.

Wireframe Development: Planning the app's structure and user interface before visual design implementation.

Iterative Refinement: Before-and-after views demonstrating key UI improvements for enhanced usability.

Visual Inspiration

The aesthetic aimed for a "FRESH + SIMPLE + MINIMALIST and FRIENDLY" feel, utilizing a clean palette of greens, blues, and pinks for an inviting and modern user experience.

Inspirational Moodboard

Design System

The Solution

Swap Now offers a community-focused digital space for second-hand clothing exchange. Key features include:

  • Group-centric trading: Facilitates exchanges within trusted circles.
  • Direct communication: Users can chat and arrange swaps/sales directly.
  • Intuitive UI: Designed to be fresh, simple, minimalist, and friendly, ensuring ease of use.

Outcomes & Learnings

The project successfully designed a platform to address the need for affordable clothing through community-driven exchanges. A key learning was the high value users place on trust and community in second-hand transactions. The established core peer-to-peer exchange functionality serves as a strong foundation for integrating future features like community events.
